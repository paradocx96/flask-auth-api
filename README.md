## Usage

### Create virtual environment

#### Windows

```bash
py -3 -m venv venv
```

#### Linux/MaxOS

```bash
python3 -m venv venv
```

### Activate virtual environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux/MaxOS

```bash
venv/bin/activate
```

### Install required libraries

```bash
pip3 install -r requirements.txt
```

### Run app locally

```bash
flask run
```
